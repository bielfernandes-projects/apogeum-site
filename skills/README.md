# Apogeum Digital — Site Institucional

Site institucional da **Apogeum Digital**, agência especializada em criação de sites de alta performance para empresas em ascensão.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS** + Shadcn UI
- **TypeScript**

## Comandos

```bash
npm run dev      # Inicia servidor de desenvolvimento
npm run build    # Gera build de produção
npm run start    # Inicia servidor de produção
npm run lint     # Verifica lint
```

## Estrutura

```
src/
  app/           # Páginas e layout
  components/    # Componentes da UI
    ui/          # Componentes Shadcn (Button, Card, Badge)
  data/          # Dados centralizados da agência
  lib/           # Utilitários
```

## Design System

- **Background**: Deep Navy Blue (#0F172A)
- **Destaques**: Pale Gold (#D4AF37 / hsl 43 74% 66%)
- **Tipografia**: Plus Jakarta Sans (headings) + Inter (body)
- **Estilo**: Dark premium, minimalista, focado em conversão
